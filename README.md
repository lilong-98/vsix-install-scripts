# VSIX Install Scripts

This repository contains public one-click install scripts for VSIX-compatible tools.

## Included scripts

- `install/codex.sh`
- `install/codex.ps1`
- `install/claude-code.sh`
- `install/claude-code.ps1`
- `install/hermes.sh`
- `install/openclaw.sh`

## Usage

```bash
curl -fsSL https://raw.githubusercontent.com/lilong-98/vsix-install-scripts/main/install/codex.sh | bash
```

Or replace `codex.sh` with the script you want to run.

## Notes

- These scripts are copied from the private docs repo so they can be fetched publicly.
- The docs repo stays unchanged.
